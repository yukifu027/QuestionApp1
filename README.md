# 概要
写真に映ったものが人間かどうかを回答する**クイズアプリ**です。ユーザーが回答後、正答数が表示され、今までの最高得点が更新されます。

# 制作背景
アプリを作成しながら、条件分岐・UserDefaults・音声ファイルの実装方法を身に付けるため。
